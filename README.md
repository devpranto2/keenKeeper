# FriendCircle - Relationship Manager

## Project Name
**FriendCircle**

## Short Description
A web application that helps you track and nurture your friendships by logging calls, texts, and video interactions. Never lose touch with people who matter most.

## Technologies Used
- Next.js
- React
- JavaScript
- Tailwind CSS
- DaisyUI
- React Toastify
- React Icons
- Recharts


## Key Features

### 1. Friend Management Dashboard
View all friends with their status (on-track, almost due, overdue), days since last contact, and quick access to detailed profiles.

### 2. Interaction Timeline
Log calls, texts, or video calls with friends. View all interactions in chronological order with filter options by interaction type.

### 3. Visual Analytics Dashboard
See donut chart visualization of your interaction patterns with real-time statistics showing call, text, and video distribution.
